# v1.0.0

初の安定版リリース。CLI タスク管理ツールとしての基本機能が安定した。

## Changes

- 曜日名「〜曜日」形式（月曜日〜日曜日）に対応 (#33)
- `--sort` オプションの複数指定に対応（`--sort project --sort due`）(#58)

## Highlights (since 0.2.0)

- **タスク管理**: add / done / close / edit（フラグ・インタラクティブ両対応）
- **一覧表示**: list（ソート、フィルタ、カラー表示、ターミナル幅対応）
- **通知**: notify（期限切れ・リマインド通知、cron 連携向け）
- **ファジー日付**: 今日、明日、曜日名、来週、来月などの自然言語入力
- **リマインド**: タスクごとに複数のリマインド日付を登録可能
- **SQLite**: 軽量で信頼性の高いローカルデータストア
